# LookUp is a up & coming browser that puts basic necessities first, keeping everything simplistic.

With how chaotic the web has become between frequent corporatism & advertisements, individuality has little space to shine. We're looking to change that with our developing web browser, LookUp. Made with Qt & Python, being worked on by 2 people (so far), we're seeking to make browsing the web a more simple, hassle-free experience. 

Out of all the web browsers on the internet, we are hoping to bring the core essentials back into browsing, starting with:
- No shared accounts
- No plugins (we're going to be providing a built-in ad-blocker already, don't worry)
- No shared data, 3rd party statistics, or any unnecessarily sharable personal data
- A FOSS approach to browsers, akin to Firefox's ideology to maintaining a free & open web
- Absolutely 0 AI included in our software (We don't find it necessary or proper to our goals)
- Encrypted passwords
- FTP capable (No promises, but we're still debating on this one)
- An entirely self-made browser engine
- A browser theme inspired by Catppuccin's amazing palletes & color theory
